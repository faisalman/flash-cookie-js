# FlashCookie.js

JavaScript interface library to manage persistent cross-browser Flash cookies.

# Example

Not yet working, maybe someday...

```js
var cookie = new FlashCookie();
cookie.set("key", "value");
cookie.get("key");              // "value"
cookie.remove("key");
cookie.get("key");              // null
```
